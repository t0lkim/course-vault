# Changelog

## 1.0.0 -- 2026-03-01

### Initial release

- Arc browser cookie extraction with AES-128-CBC decryption (v10 nonce+IV format)
- Teachable JSON API integration for curriculum enumeration
- Direct CDN video download via yt-dlp
- MP4 metadata embedding via ffmpeg (title, artist, album, track)
- `course-info.json` manifest with full course structure and attachment metadata
- Verbose mode (`-v`) for debugging
- Resume support (skips existing files)
